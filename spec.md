# アルバイト給与管理アプリ 要件定義書

## 概要

大学生・主婦層のアルバイト収入を管理するスマートフォンアプリ。掛け持ちバイトに対応し、扶養の壁までの残り枠をリアルタイムで把握できる。2026年以降の税制改正（178万円の壁）にも対応し、税制ニュースの周知機能も持つ。

### ターゲットユーザー
- 大学生（掛け持ちバイト、親の扶養内で働きたい）
- 主婦層（配偶者の扶養内で働きたい）

---

## 技術スタック

| レイヤー | 採用技術 |
|---|---|
| フロントエンド | React Native + Expo GO |
| 言語 | TypeScript |
| ルーティング | Expo Router |
| ローカルDB | SQLite（expo-sqlite） |
| 税制ルール管理 | GitHub JSON（起動時fetch） |
| 課金 | RevenueCat（月額200円） |

### 設計方針
- **データはすべてローカル保存**。ユーザーの収入情報をサーバーで管理しない
- 複数端末同期は対象外
- 税制の数値（壁の金額）はアプリにハードコードせず、GitHubのJSONファイルで管理し、起動時にfetchして更新する

## データモデル

### `user_settings`（ユーザー設定）
| カラム | 型 | 説明 |
|---|---|---|
| birth_date | TEXT | 生年月日（YYYY-MM-DD） |
| dependent_type | TEXT | `parent` / `spouse` / `none` |
| large_company | BOOLEAN | 勤務先が大企業か（106万円壁の判定用） |
| carryover_income | INTEGER | 年初繰越収入（円）。年途中インストール対応 |
| plan | TEXT | `free` / `premium` |

### `jobs`（バイト先）
| カラム | 型 | 説明 |
|---|---|---|
| id | INTEGER PK | |
| name | TEXT | 店名・会社名 |
| hourly_wage | INTEGER | 時給（円） |
| employment_type | TEXT | `part` / `dispatch` / `other` |
| is_active | BOOLEAN | 現在も勤務中か |

### `shifts`（勤務記録）
| カラム | 型 | 説明 |
|---|---|---|
| id | INTEGER PK | |
| job_id | INTEGER FK | |
| date | TEXT | 勤務日（YYYY-MM-DD） |
| start_time | TEXT | 開始時刻（HH:MM） |
| end_time | TEXT | 終了時刻（HH:MM） |
| break_minutes | INTEGER | 休憩時間（分） |
| estimated_wage | INTEGER | 見込み給与（自動計算） |

### `payslips`（給与明細）
| カラム | 型 | 説明 |
|---|---|---|
| id | INTEGER PK | |
| job_id | INTEGER FK | |
| year | INTEGER | 対象年 |
| month | INTEGER | 対象月（1〜12） |

> **収入集計のルール**: `payslips`の実績値を優先。まだ明細が届いていない月は`shifts`の見込み値で補完する。

## 扶養壁の判定ロジック（2026年基準）

オンボーディングの回答からロジックで壁を自動判定する。ユーザーに「どの壁を気にしますか？」と聞かない。

### 判定フロー

```
1. dependent_type == "none"
   → 壁なし（所得税178万円のみ表示）

2. dependent_type == "parent"
   - 年齢が19〜22歳（特定扶養）
     → 社保の扶養: 150万円
     → 扶養控除（親の税金）: 150万円
     → 自分の所得税: 178万円
   - 年齢が23歳以上（一般扶養）
     → 社保の扶養: 130万円
     → 扶養控除（親の税金）: 178万円
     → 自分の所得税: 178万円

3. dependent_type == "spouse"
   → 社保の扶養: 130万円
   → 配偶者控除: 178万円
   → 自分の所得税: 178万円

4. large_company == true かつ 週20時間以上勤務
   → 上記に加えて106万円の壁を表示
   → ※2026年10月以降、賃金要件撤廃予定
```

### GitHub JSON 構造（tax_rules.json）

```json
{
  "version": "2026",
  "updated_at": "2026-01-01",
  "walls": {
    "income_tax": 1780000,
    "dependent_general": 1780000,
    "dependent_specific_age_min": 19,
    "dependent_specific_age_max": 22,
    "dependent_specific_limit": 1500000,
    "social_insurance_basic": 1300000,
    "social_insurance_large_company": 1060000,
    "residence_tax_approx": 1100000
  },
  "news": [
    {
      "id": "2026-reform-178",
      "title": "178万円の壁が2026年分から適用",
      "body": "所得税の非課税枠が178万円に拡大されました。ただし社会保険の扶養（130万・150万円）は別制度です。",
      "published_at": "2026-01-01",
      "important": true
    }
  ]
}
```

## 機能一覧

### 無料機能

| 機能 | 説明 |
|---|---|
| オンボーディング | 生年月日・扶養種別・勤務先規模・繰越収入を入力し、適用される壁を自動判定 |
| バイト先登録 | 複数のバイト先を登録（店名・時給・雇用形態） |
| シフト入力 | 勤務日・開始/終了時刻・休憩時間を入力。見込み給与を自動計算 |
| 給与明細入力 | 実際の支給額を入力し、見込みとの差分をダブルチェック |
| 残り枠表示（ホーム） | 年間残り金額・残り時間（時給換算）・壁まで何％かのゲージ |
| 月次グラフ（過去） | バイト先別・月別の収入実績グラフ |
| 税制ニュース通知 | GitHub JSONから取得した最新の税制情報をアプリ内で表示 |

### 有料機能（月額200円 / RevenueCat）

| 機能 | 説明 |
|---|---|
| 収入予測グラフ | 現在のペースをもとに年末までの収入推移を予測・グラフ表示 |
| ペース警告 | 「このペースで働くと〇月に壁を超えます」のアラート |
| 確定申告サポート | 年間収入の集計、申告要否の判定、e-Tax入力用の数字表示・コピー、e-Taxへのディープリンク |

---

## 確定申告サポートの詳細

- **対象**: 2か所以上からの給与があり、年末調整されない副収入が年20万円超のユーザー
- **提供内容**:
  1. バイト先別・年間収入の集計表示
  2. 「確定申告が必要か」の自動判定
  3. e-Tax入力に必要な数字をコピーしやすい形で表示
  4. e-Taxアプリ（国税庁）へのディープリンク誘導
- **提供しないもの**: API連携による自動申告、PDF書類の生成

## 注意事項・免責

- 本アプリの計算はあくまで目安です。正確な判定は税理士・社会保険労務士にご相談ください
- 税制は毎年変わります。アプリ内の情報は最新でない場合があります
- 住民税の非課税ラインは自治体によって異なります（目安: 110〜120万円）
